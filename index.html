1. What is it and Why is it important?


When we perform in UI/UX design, or in graphics rendering we sometimes want to animate things over time. For this, what we do is calculate instead of resorting to a complete physical simulation. For example, when moving an object from one position to another, fade a screen by changing the alpha.

2. What is a spline?

In mathematics it is a piecewise function defined by polynomials. Specifically they are used in programming due to their ease and precision of evaluation.

3. Spline usability

We can use them in camera movement in 2D video games, for example in the Sonic the Hedgehog game, where all the loops and transitions within a level are created through a spline. Also in 3D video games it is used to track the character. Other uses they may have is in the design of all interface objects that exhibit temporary, ease behavior. Flexibility creates and reinforces the naturalness inherent in the fluidity of user experiences and creates a sense of continuity when objects behave as users expect them to.

4. Easing Functions

Before we can do it first we need to know the four values that we will use later.
Initial Value (i_pos): It is the initial value from which we want to create the animation, if we want to move it it will be the initial position if we want to increase it it will be the scale.
Final Value (f_pos): It is the final value from which we want to create the animation, if we want to move it it will be the initial position if we want to increase it it will be the scale.
Duration (duration): It is the value of the time which we want the animation to last.
Total elapsed time (time_passed): This is the total time that has passed since the spline started.

5. Splines Types

Ease Out: Accelerated at the beginning and slow at the end.
